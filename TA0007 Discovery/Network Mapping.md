## DNS Configuration
```
ipconfig /displaydns

# View DNS settings 
nslookup 
server 
set type=all

# Clear DNS cache 
ipconfig /flushdns 

# View hosts file entries 
type C:\Windows\System32\drivers\etc\hosts
```
